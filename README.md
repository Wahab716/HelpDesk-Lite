# HelpDesk Lite

HelpDesk Lite is a full-stack IT ticketing and support dashboard.

Employees can submit support tickets, and support/admin users can manage tickets by updating status, setting priority, leaving comments, and viewing dashboard stats.

## Features

* User registration and login
* Create support tickets
* View and manage tickets
* Update ticket status
* Set ticket priority
* Add ticket comments
* Search and filter tickets
* Dashboard statistics

## Project Goal

The goal of HelpDesk Lite is to build a realistic full-stack application that demonstrates frontend development, backend APIs, database integration, authentication, and deployment.
